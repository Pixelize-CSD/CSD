Code War

Event Description

Code War, a debugging showdown exclusively for C/C++ enthusiasts, challenges participants to conquer logical errors within provided code snippets. Armed with their programming prowess, participants delve into the intricate world of debugging to identify and fix logical flaws. This competition sharpens problem-solving skills while focusing solely on the C/C++ languages


Date: 24.02.24

Duration: 15 mins

Venue: CSD Block


About the Event

C is a procedural programming language developed by Dennis Ritchie at Bell Labs in the early 1970s. It was designed to provide low-level access to system memory and language constructs that map efficiently to machine instructions. C has a simple syntax and is widely used for system programming, embedded systems, and operating systems development. It has influenced many other programming languages and is often referred to as the "mother" of all programming languages.


What to Expect

  • The code with logical errors will be given.
  • The code should be fixed within a given time.
  • Least modification to the code will act as a bonus.
